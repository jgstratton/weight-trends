# Weight Trends

A self-hosted web app for tracking body weight and visualising trends over time.  
Passwordless login (email OTP) · PostgreSQL · SvelteKit · Docker Compose · Chart.js

---

## Features

- **Passwordless login** – enter your email, receive a 6-digit one-time code, done.
- **Weight logging** – log a weight entry with date and optional notes. Edit or delete any entry.
- **Trend chart** – line chart of raw data plus a 7-day simple moving average.
- **Self-hosted** – runs entirely in Docker; no external services required beyond an email provider.

## Prerequisites

| Tool | Minimum version |
|---|---|
| Docker | 24+ |
| Docker Compose | v2 (bundled with Docker Desktop) |

You also need a [Resend](https://resend.com) account to send the OTP emails (free tier is fine).

## Quick start

### 1. Clone the repository

```sh
git clone https://github.com/your-username/weight-trends.git
cd weight-trends-2
```

### 2. Configure environment variables

```sh
cp .env.example .env
```

Edit `.env` and fill in your values:

```dotenv
# Resend API key – https://resend.com/api-keys
RESEND_API_KEY=re_xxxxxxxxxxxxxxxxxxxxxxxxxxxx

# Sender address – must be a verified domain/address in your Resend account
RESEND_FROM_EMAIL=Weight Trends <noreply@yourdomain.com>
```

> The `DATABASE_URL` is wired up automatically via Docker Compose and does **not** need to be set in `.env`.

### 3. Build and start

```sh
docker compose up --build
```

The app will be available at **http://localhost:3000**.

On first start, the entrypoint automatically runs database migrations before the server comes up.

### 4. Log in

Open http://localhost:3000, enter your email address and submit the form.  
Check your inbox for the 6-digit code, enter it, and you're in.

## Development (local, without Docker)

### Prerequisites

- Node.js 22+
- A running PostgreSQL instance (or use the Docker Compose Postgres service on its own)

### Setup

```sh
cd app
npm install
```

Create `app/.env` (used by Vite's `$env/static/private`):

```dotenv
DATABASE_URL=postgres://weighttrends:weighttrends@localhost:5432/weighttrends
RESEND_API_KEY=re_xxxxxxxxxxxxxxxxxxxxxxxxxxxx
RESEND_FROM_EMAIL=noreply@yourdomain.com
```

Start just the database:

```sh
docker compose up postgres -d
```

Run migrations:

```sh
cd app
npx drizzle-kit migrate
```

Start the dev server:

```sh
npm run dev
```

Open http://localhost:5173.

## Configuration reference

| `DATABASE_URL` | Yes | PostgreSQL connection string. Set automatically in Docker Compose. |
